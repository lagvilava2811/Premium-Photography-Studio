# Anuka Abazadze Photography Studio

Premium photography portfolio website featuring an elegant **Obsidian & Champagne** design aesthetic.

## ✨ Features

- 🎨 Luxurious dark theme with gold accents
- 📸 Interactive 3D WebGL photo gallery with floating cards
- 🗓️ Online booking system with calendar
- 🖼️ Portfolio with 3D dodecagon carousel
- 📱 Fully responsive design
- 🔐 Admin dashboard for managing bookings, gallery & SEO
- ✉️ Contact form with service selection

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Three.js for 3D WebGL effects
- Google Fonts (Inter)
- No frameworks — pure performance

## 📁 Structure

```
├── index.html          # Homepage
├── about.html          # About page
├── portfolio.html      # Portfolio gallery
├── services.html       # Services & pricing
├── blog.html           # Blog page
├── contact.html        # Contact & booking
├── admin.html          # Admin dashboard
├── css/                # Stylesheets
├── js/                 # JavaScript modules
└── assets/             # Images & media
```

## 🚀 Getting Started

Simply open `index.html` in your browser.

## 👤 Admin Access

Navigate to `admin.html` to manage your portfolio.
